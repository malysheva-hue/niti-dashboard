# Нити Творчества — Дашборд

Внутренний дашборд отдела продаж бренда «Нити Творчества» на Wildberries.

## Запуск

Открыть `index.html` двойным кликом. Никакого сервера не нужно — все данные встроены в HTML.

Пароль на входе — у РОПа.

## Что внутри

- `index.html` — единый файл со встроенными данными за 12 месяцев и всеми зависимостями (Plotly/SheetJS/marked по CDN).
- `build.py` — собирает HTML и `data.json` из локальных CSV/XLSX. Источники данных в репо не хранятся.
- `data.json` — общий лёгкий срез (KPI, OOS, цены, сводка дня) для отладки и будущей подгрузки через fetch.

## Этапы

- ✅ Этап 1 — статический MVP (текущий).
- ⏳ Этап 2 — `update.py` для подгрузки свежих данных.
- ⏳ Этап 3 — БД + карточки SKU + история по дням.
- ⏳ Этап 4 — личные кабинеты с авторизацией.
- ⏳ Этап 5 — виджет формирования поставок на ВБ.
- ⏳ Этап 6 — AI-рекомендации через Claude API.
